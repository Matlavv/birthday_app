## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

then run

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Custom tailwind.config.ts for custom color.
next/fonts to get the fonts.
